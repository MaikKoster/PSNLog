# Get-NLogLogger

## SYNOPSIS
Gets a NLog Logger instance

## SYNTAX

```
Get-NLogLogger [[-LoggerName] <String>] [<CommonParameters>]
```

## DESCRIPTION
The Get-NLogLogger Cmdlet gets the specified NLog logger.
