# Write-NLogVerbose

## SYNOPSIS
Writes a message on 'Debug' log level and to the Verbose Message stream.

## SYNTAX

```
Write-NLogVerbose [-Message] <String> [<CommonParameters>]
```

## DESCRIPTION
Writes a message on 'Debug' log level and to the Verbose Message stream.
Can be used to override the native Write-Verbose CmdLet.
